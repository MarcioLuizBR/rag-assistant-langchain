# 🤖 RAG Assistant com LangChain

Projeto de estudo e aplicação prática de um sistema de IA baseado em RAG (Retrieval-Augmented Generation), utilizando Python e LangChain.

O objetivo é construir um assistente capaz de responder perguntas com base em documentos, utilizando embeddings e um banco vetorial para recuperação de contexto relevante.

---

## 🚀 Tecnologias utilizadas

- Python
- LangChain
- Embeddings (OpenAI ou similar)
- Vector Database (Chroma / FAISS)
- Manipulação de PDFs

---

## 🧠 Conceitos aplicados

- RAG (Retrieval-Augmented Generation)
- Embeddings
- Busca semântica
- Banco vetorial
- Chunking de texto
- Integração com LLM
